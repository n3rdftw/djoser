JWT Endpoints
=============

JWT Create
----------

Use this endpoint to obtain JWT.

**Default URL**: ``/jwt/create/``

+----------+---------------------------------+----------------------------------+
| Method   |           Request               |           Response               |
+==========+=================================+==================================+
| ``POST`` | * ``token``                     | ``HTTP_200_OK``                  |
|          |                                 |                                  |
|          |                                 | * ``token``                      |
|          |                                 |                                  |
|          |                                 | ``HTTP_400_BAD_REQUEST``         |
|          |                                 |                                  |
|          |                                 |  * ``non_field_errors``          |
+----------+---------------------------------+----------------------------------+
